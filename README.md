# 🔗 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a friendship management app that helps you stay connected with the people who matter most. Track when you last reached out, log interactions, and get reminded before friendships go cold.

---

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| React.js | Build the UI |
| React Router DOM | Handle page navigation |
| Tailwind CSS | Styling and responsiveness |
| Recharts | Friendship analytics pie chart |
| React Hot Toast | Toast notifications |

---

## ✨ Features

- 📋 **Friend Dashboard** — View all your friends as cards with their status, tags, and days since last contact
- 📜 **Interaction Timeline** — Automatically logs every Call, Text, and Video interaction with date and type
- 📊 **Friendship Analytics** — Visual pie chart breaking down your interaction history by type

## 🏃 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/keenkeeper.git

# Navigate into the project
cd keenkeeper

# Install dependencies
npm install

# Start the development server
npm run dev
```
